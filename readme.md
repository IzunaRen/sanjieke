# 三节课视频自动播放脚本

## 功能介绍

这是一个用于「三节课」(sntelelearning.b.sanjieke.cn) 网站的视频自动播放脚本，可实现以下功能：

- 🎬 **自动播放控制**：自动触发视频播放按钮
- ⚡ **倍速设置**：自动将视频倍速调整为1.25x
- ⏭️ **自动续播**：当前视频播放完成后，自动播放下一个视频
- 📚 **章节自动切换**：当前章节视频全部播放完成后，自动切换到下一章节
- 🎯 **浮动控制按钮**：提供可拖动的浮动按钮，方便随时控制脚本
- 🔄 **状态监控**：实时监控视频播放状态和节点完成状态

## 安装方法

### 1. 安装浏览器扩展

该脚本需要通过 Tampermonkey 或 Greasemonkey 等用户脚本管理器运行：

- Chrome/Firefox/Edge: [Tampermonkey](https://www.tampermonkey.net/)
- Firefox: [Greasemonkey](https://addons.mozilla.org/zh-CN/firefox/addon/greasemonkey/)

### 2. 添加脚本

1. 打开 Tampermonkey 扩展
2. 点击「添加新脚本」
3. 复制本脚本的完整代码
4. 粘贴到编辑器中并保存

## 使用说明

### 1. 启动脚本

1. 访问三节课的视频播放页面
2. 页面右上角会出现一个绿色的浮动按钮
3. 点击该按钮启动自动播放功能

### 2. 按钮功能

- **点击按钮**：启动自动播放、初始化视频、设置倍速、开始状态监控
- **拖动按钮**：按住按钮可拖动到页面任意位置
- **状态指示**：点击后按钮变为蓝色渐变并显示动画效果，表示脚本正在运行

### 3. 自动操作流程

1. 点击浮动按钮后，脚本会：

   - 自动触发视频播放按钮
   - 查找并初始化视频元素
   - 设置视频倍速为1.25x
   - 开始监控视频播放状态
   - 开始监控节点完成状态（每30秒检查一次）
2. 视频播放过程中：

   - 每秒检查一次视频播放状态
   - 每30秒检查一次当前节点完成状态
   - 如果视频暂停，会自动恢复播放
3. 视频播放完成后：

   - 检查当前页面是否还有其他视频
   - 如果有，自动播放下一个视频
   - 如果没有，尝试切换到下一章节的第一个视频
   - 所有视频播放完成后，显示完成提示

## 技术实现

### 核心功能模块

#### 1. 浮动按钮管理

- 创建可拖动的浮动按钮
- 实现按钮点击、拖动、状态切换等交互
- 使用 SVG 图标增强视觉效果

#### 2. 视频控制

- 自动查找并触发页面上的播放按钮
- 查找视频容器和视频元素
- 设置视频播放倍速
- 监控视频播放状态

#### 3. 节点管理

- 查找当前活跃的视频节点
- 检查节点完成状态
- 实现章节自动切换逻辑

#### 4. 状态监控

- 视频播放状态监控（每秒检查一次）
- 节点完成状态监控（每30秒检查一次）
- 异常情况处理和重试机制

### 关键函数说明

#### `createPlayButton()`

创建浮动可拖动的播放触发按钮

#### `triggerPlayButtons()`

触发页面中所有的播放按钮

#### `setPlaybackRateTo125()`

设置视频倍速为1.25x

#### `initTargetVideo()`

初始化目标视频元素

#### `isPlaying()`

判断视频是否正在播放

#### `checkNodeStatus()`

检查当前节点完成状态

#### `startPlayStatusCheck()`

启动视频播放状态检查

#### `startNodeStatusCheck()`

启动节点状态检查

#### `checkPlayStatus()`

检查视频播放状态，完成则自动续播

#### `nextNode()`

跳转到下一个节点或章节

## 故障排除

### 1. 按钮不显示

- 检查 Tampermonkey 是否已启用
- 刷新页面重试
- 确认脚本是否正确安装

### 2. 视频不自动播放

- 确保已点击绿色浮动按钮启动脚本
- 检查视频容器是否加载完成
- 尝试刷新页面后重新启动脚本

### 3. 倍速设置失败

- 脚本会自动重试设置倍速
- 可手动设置倍速后继续使用其他功能
